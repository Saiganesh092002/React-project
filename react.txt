what is React.js
=================
-->It is a library for java-script.
-->React.js is used to develop only single page application.
-->React.js is a component based ( components are re-usable ).
-->React.js will renders/update data on the web-page without a page re-loading.
-->React can be used for both client-side and server side.

features of React.js
====================
1)it is declarative
   (it will renders any data on the web-page).
2)it is component based
3)learn once write anywhere.

how to create React application
==================================
-->ways of achiving React application
  1)by using CDN way  ---> will be used in production stage
  2)by using node.js (npm , npx) way

2)by using node.js way
=======================
two packages:-
1)NPM (node package manager)
-->npm is used to install , update and delete an package in java-script.
-->in npm packages should be installed globally.

eg:- npm install -g create-react-app 
     create-react-app  project-name

note :- create-react-app is deprecated.

2)NPX (node package executor)
==============================
-->npx is used to execute the packages without intsalling.
-->in npx no need of using globally.

eg:-

npx create-react-app project-name

-->it will create react-17 to react-18

command to create react-19 application

npm --> used to install all the node packages.
